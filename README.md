# most-used-text

A simple app that count the most used text Made with Golang

---

### Running the App

```sh
// clone the project
git clone https://github.com/thirthfamous/most-used-text/

// go to the project directory
cd most-used-text

// install the dependencies
go get

// run the tests
go test \test
or
go test .\test\

// run the main
go run main.go
```
